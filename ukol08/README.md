# Úkol08 - Minutka

Program při startu/resetu nastaví čas odpočtu na 2 minuty. Přes dotykový displej je možné tuto hodnotu zvýšit/snížit o 10 sekund.

Na displeji se stiskem bloku ,,SET'' aktivuje časovač a na displeji se zobrazuje zbývající čas do alarmu. Po uplynutí zadané doby celý displej zčervená.

Tlačítkem se odpočet vypne a nastaví výchozí hodnota.
